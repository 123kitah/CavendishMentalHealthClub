# Cavendish Mental Health Club - Original Assignment Questions

## Question 1: Registration Screen Design (10 Marks)

### a) Layout Choice
**Question:** Which layout type did you choose for the Registration Screen? Explain why you chose this layout type and what alternatives you considered.

### b) Input Fields and Kotlin Variables
**Question:** List all the input fields you included in your Registration Screen. For each input field, specify:
- The type of input field
- The Kotlin variable(s) used to store the input data
- The data type of the variable(s)

### c) Capturing Input When Register Button is Tapped
**Question:** Explain the process of how you capture the user input when the Register button is tapped. Describe the steps involved, including:
- How the button click is handled
- How input is captured from each field
- Any validation performed
- How the data is prepared for navigation to the next screen

## Question 2: Confirmation Screen Design (10 Marks)

### a) Creating a Kotlin Class to Hold User Data
**Question:** Show the Kotlin class you created to hold the user data. Explain:
- Why you chose this type of class (data class, regular class, etc.)
- What properties the class contains
- Why the class implements Parcelable (or the interface you chose)

### b) Sending Data Using Intent
**Question:** Explain how you send the user data from the Registration Screen to the Confirmation Screen using an Intent. Include:
- The code you used to create the Intent
- How you added the data to the Intent
- The method used to start the new activity

### c) Receiving and Displaying Data on Confirmation Screen
**Question:** Explain how you receive and display the user data on the Confirmation Screen. Describe:
- How you retrieve the data from the Intent
- How you display the data in the layout
- The layout structure you used to organize the information
